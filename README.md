# Quantum Billionaire 🎲💰

一款基于Pygame开发的多人策略棋盘游戏，以校园为背景的财富积累竞技游戏。

## 🎮 游戏概述

Quantum Billionaire是一款融合了传统大富翁玩法和现代小游戏元素的策略游戏。玩家扮演校园中的学生，通过在不同的校园地点间移动，参与各种活动和挑战，最终目标是在有限的回合内积累最多的财富。

## ✨ 游戏特色

- **校园主题**：以真实校园地点为背景，包含宿舍区、教学楼、体育馆等熟悉场所
- **多样化小游戏**：集成了Simon记忆游戏、量子炸弹等趣味小游戏
- **策略性投资系统**：通过智慧的投资决策积累财富
- **回合制竞技**：公平的回合限制确保游戏节奏
- **精美界面**：直观的游戏界面和流畅的动画效果

## 🎯 游戏玩法

### 基本规则

1. **游戏目标**：在规定回合数内积累最多财富的玩家获胜
2. **玩家数量**：3名玩家（红色小人、橙色小人、蓝色小人）
3. **回合限制**：默认20回合（可在设置中调整）

### 游戏流程

#### 🎲 掷骰子阶段
- 点击骰子进行投掷
- 根据点数在地图上移动相应步数
- 到达新地点触发相应事件

| 🔬 理科楼 | 科学实验室 | 小游戏挑战 |
| 🏛️ 工字厅 | 行政办公区 | 投资机会 |
| 🎭 大礼堂 | 文艺活动中心 | 特殊事件 |

| 📚 清华学堂 | 学术殿堂 | 知识挑战 |

#### 🎮 小游戏系统
- **Simon记忆游戏**：考验玩家记忆力和反应速度
- **量子炸弹**：基于量子力学概念的策略游戏
- **其他挑战**：多样化的互动小游戏

#### 🛒 商店系统
- 购买道具增强实力
- 投资获得长期收益
- 策略性资源管理

### 💰 财富系统

- **初始资金**：每位玩家开始时拥有$1000
- **收入来源**：
  - 小游戏奖励
  - 投资收益
  - 特殊事件奖励
- **支出项目**：
  - 商店购物
  - 投资成本
  - 事件惩罚

## 🎲 游戏操作

### 鼠标控制
- **左键点击骰子**：掷骰子移动
- **左键点击按钮**：确认选择/进入下一阶段
- **左键点击商店**：进行购买操作

### 界面元素
- **回合显示**：屏幕右上角显示当前回合进度
- **信息面板**：显示当前玩家状态和游戏信息
- **地图路径**：虚线箭头指示移动路径

## 🏆 胜利条件

游戏在达到最大回合数后结束，按以下标准判定胜负：

1. **第一名**：拥有最多财富的玩家
2. **第二名**：财富次多的玩家
3. **第三名**：财富最少的玩家

游戏结束后会显示：
- 最终排名
- 各玩家财富总额
- 重新开始选项

## ⚙️ 游戏设置

### 可调整参数
- `max_rounds`：最大回合数（默认20）
- `screen_width/height`：窗口尺寸
- 初始金钱数量
- 小游戏难度

### 配置文件
- `locations_data_path`：地点信息配置
- `events_path`：事件数据配置

## 🎨 自定义扩展

### 添加新地点
1. 在`location.py`中创建新的地点类
2. 在`locations_data_path`文件中添加坐标和信息
3. 在`create_location()`函数中添加判断逻辑

### 添加新小游戏
1. 在`mini_games/`目录下创建新的游戏模块
2. 实现`play()`函数返回游戏结果
3. 在`run_specific_mini_game()`中添加调用逻辑

### 修改游戏平衡
- 调整初始金钱数量
- 修改小游戏奖励数值
- 改变回合数限制

## 🐛 故障排除

### 常见问题

**Q: 游戏启动时黑屏**
A: 检查是否正确安装pygame库，确认资源文件路径正确

**Q: 小游戏无法启动**
A: 验证小游戏模块是否完整，检查import语句

**Q: 游戏卡死在某个状态**
A: 检查游戏状态转换逻辑，确认所有按钮响应正常

### 性能优化
- 建议使用SSD硬盘提高资源加载速度
- 关闭不必要的后台程序释放内存
- 调整游戏帧率适应硬件性能
